# OxygenNotes

**Version: B31 [BETA]**

---

## Philosophy

OxygenNotes is a notes app with broad functionality that stays clean and uncluttered.
The goal: something that doesn't look like an engineering tool, has modern Material Design,
and centralizes ideas in one place — folders, drawings, whiteboards, todos, all together.

**Core principle: every feature must feel necessary. Nothing decorative, nothing redundant.**

---

## Target Feel

- Modern, polished Material Design 3
- Folder-first organization (not tag-first, not search-first)
- Centralized — user should not need another app for sketches, todos, or whiteboards
